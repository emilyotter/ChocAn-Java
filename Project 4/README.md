# Final Project CS 200

**To build and run:**
``` shell
# Delete possible leftover build files
ant cleanall
# Compile the class files
ant compile
# Generate the .jar file
ant jar
# Clean up build files
ant clean
# Run the jar executable
ant run 
```

**Alternatively:**
``` shell
ant cleanall compile jar clean run
```
